## Frontend (Next.js)

This is the web app.

Tech:
- Next.js 14
- Chakra UI
- React Query

### How to run (dev)

1) Install deps in repo root:
- pnpm install

2) Start frontend:
- pnpm --filter frontend dev

Open http://localhost:3000

### Notes
- The app talks to the backend at /api.
- Auth uses cookies.
